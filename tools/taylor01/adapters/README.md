# Taylor01 Adapters

Reserved for reusable contracts and implementations for third-party systems such as Linear, GitHub, Discord, Slack, Jira, Hermes, OpenAI runtimes, and historical OpenClaw projections.

Current Vera adapter surfaces:

- `hermes/` = Hermes-first runtime coordination contract
- `hermes/vera/` = Hermes-first Vera execution-path contract
- `openai/vera/` = current bridge execution adapters
- `openclaw/vera/` = historical OpenClaw closure/mapping residue only

Do not add new OpenClaw fallback or alternate-runtime assumptions here.
